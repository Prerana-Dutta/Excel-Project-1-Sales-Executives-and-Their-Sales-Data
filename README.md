# Excel-Project-1-Sales-Executives-and-Their-Sales-Data
An interactive dashboard dummy project that analyzes sales executives’ performance across regions, highlighting the top and bottom performers and providing clear visual insights to support decision‑making.


##H2 Project Overview
This project delivers insights into sales executives and their target achievements across different regions. It was designed to highlight performance extremes and provide an interactive way to explore regional sales data.

Objectives
Track and compare sales executives’ performance across regions.
Identify the top five and bottom five performers for quick benchmarking.
Build an interactive dashboard that allows seamless navigation and control.
Provide clear visualisations to support decision‑making.
Data Preparation
Data preparation involves eliminating duplicate entries and restructuring the dataset into a usable format, ensuring it is clean and ready for analysis.
Analysis Methods
Charts: Bar charts, pie charts, and area charts to visualise performance metrics.
Region Slicer: Enabled users to switch between regional views effortlessly.
Developer Tool Checkboxes: Allowed users to select which sales tables to display.
Macro Recording: Established report connections between slicers and tables.
VBA Code: Linked slicers with checkboxes to ensure smooth interactivity.
Key Findings
The dashboard highlights the top five performers with the highest sales and the bottom five performers with the lowest sales. This comparison makes it easy to identify both high achievers and areas needing improvement.
Regional breakdowns reveal performance variations, helping managers focus on specific regions.

 Conclusion


Miny Mole and Surendra Kumar are the top performers with the highest sales and target hit rates. 
Shailendra Barge and Kavita Sharma also show strong alignment with targets.
John D’Costa, Salim, and Rekha Jain have the lowest target achievement, below 43%.
